Add FileForge to Continue.dev

Supercharge Continue.dev with FileForge capabilities!

1
Open Continue configuration
Find your config.json file (usually in ~/.continue/)
2
Add MCP server configuration
Add FileForge to the mcpServers section
3
Restart Continue
Reload your editor to activate FileForge
{ "mcpServers": { "fileforge": { "command": "npx", "args": [ "--yes", "github:iMAGRAY/FileForge", "node", "src/fileforge.cjs" ], "env": { "NODE_PATH": "./node_modules" } } } }
